# 安装

> 注意目前是内网 alpha 阶段，可能会有一些 bug，请选择使用。

### 使用 PNPM 安装


<n-card>

```bash
pnpm add naive-ui-protable-alpha -D

```

</n-card>


### 使用 NPM 安装

<n-card>


```bash
npm install naive-ui-protable-alpha -D
```



</n-card>

### 使用 YARN 安装

<n-card>

```bash
yarn add naive-ui-protable-alpha -D
```

</n-card>
