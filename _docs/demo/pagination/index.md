# 分页

通过`pagination`属性配置 Protable 的分页器。分页器的详细属性见[Pagination Props - Naive UI](https://www.naiveui.com/zh-CN/os-theme/components/pagination#Pagination-Props)。

::: tip

- 在 Protable 中，若`sync-route`不为 false，分页器的变化将与路由同步。  
  :::
